    AI文本冒险游戏 - 最简单的运行方法
═══════════════════════════════════════════════════════

【Windows 用户】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
方法1：双击运行（最简单）
  1. 双击 "启动游戏.bat" 文件
  2. 等待启动完成
  3. 浏览器会自动打开游戏

方法2：命令行运行
  1. 打开命令提示符（CMD）或 PowerShell
  2. 进入游戏目录
  3. 运行：pip install -r requirements.txt
  4. 运行：python game_server.py
  5. 浏览器访问：http://127.0.0.1:5001


【Linux/Mac 用户】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
方法1：使用启动脚本
  1. 终端执行：chmod +x 启动游戏.sh
  2. 运行：./启动游戏.sh
  3. 浏览器访问：http://127.0.0.1:5001

方法2：命令行运行
  1. 终端进入游戏目录
  2. 运行：pip3 install -r requirements.txt
  3. 运行：python3 game_server.py
  4. 浏览器访问：http://127.0.0.1:5001


【环境要求】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  • Python 3.8 或更高版本
  • 浏览器（Chrome/Firefox/Edge）


【停止游戏】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  在运行服务器的命令行窗口中按 Ctrl+C


【常见问题】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  Q: 提示"模块未找到"？
  A: 运行 pip install -r requirements.txt 安装依赖

  Q: 端口被占用？
  A: 修改 game_server.py 最后一行的端口号（默认5001）

  Q: 浏览器无法访问？
  A: 确认服务器已启动，尝试访问 http://localhost:5001
